Destaques

Funcionalidade 1

Nosso programa Java é responsável por gerenciar a lógica de domínio de um sistema de teleatendimento médico, permitindo o cadastro e autenticação de médicos com base em suas especialidades e número de CRM. Ele serve como parte do backend do projeto, oferecendo endpoints ou funcionalidades essenciais para integração com assistentes pessoais, interfaces web e sistemas de posicionamento inteligente para pacientes idosos.

Funcionalidade 2

Este programa Python simula o menu de um app de teleatendimento médico, permitindo marcar consultas, verificar datas disponíveis, visualizar histórico e acessar links de chatbot e reconhecimento facial. Ele gerencia interações básicas do usuário, armazenando os agendamentos em memória e servindo como protótipo funcional para redução do absenteísmo entre pacientes idoso.
