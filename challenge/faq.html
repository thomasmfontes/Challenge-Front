Perguntas Frequentes

O que é o projeto?

O projeto é uma solução desenvolvida por alunos da FIAP, utilizando HTML, CSS e JavaScript puros, para resolver um problema real com design responsivo.

Como o site foi construído?

Todo o site foi feito sem frameworks, usando apenas HTML semântico, CSS externo e JavaScript puro, seguindo as regras da disciplina.

O projeto é responsivo?

Sim, o design foi adaptado para funcionar bem em celulares, tablets e desktops usando media queries.

Onde posso encontrar o repositório?

Você pode acessar o repositório pelo link que está disponível no rodapé da página ou no arquivo README.md.
